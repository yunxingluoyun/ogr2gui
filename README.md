OGR2GUI 是一个开源项目，为 OGR2OGR 提供图形用户界面 （GUI）。OGR2OGR 是用于转换和操作地理空间数据的命令行实用程序。

将原项目进行升级：

1. 其在Qt5+GDAL3.7.2能够正常编译；
2. 增加ogr2gui_zh.ts支持简体中文；
